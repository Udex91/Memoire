Memoire
Memoire is a clean and intuitive note-taking application built with React. It allows users to capture ideas, manage notes, and quickly find information through search — all within a simple, distraction-free interface.

Features
    ✔ Create and save notes instantly
    ✔ Real-time search across notes
    ✔ Delete notes with a single click
    ✔ Light / Dark mode toggle
    ✔ Automatic date tracking for each note
    ✔ Character limit with remaining count indicator
    ✔ Persistent storage using browserlocalStorage
    ✔ Responsive layout for desktop and mobile

 Built With
    ✔ React: Component-based UI development
    ✔ JavaScript (ES6+)
    ✔ CSS 
    ✔ Local Storage API 
    ✔ Create React App

Getting Started
To run Memoire locally:
1️⃣ Clone the repository
git clone https://github.com/Udex91/memoire.git
2️⃣ Navigate into the project
cd memoire
3️⃣ Install dependencies
npm install
4️⃣ Start the development server
npm start
The app will be available at:
http://localhost:3000
